# Photo Search Engine Poster

- `poster.svg`: A3 print layout with 3mm bleed-aware composition (`303mm × 426mm`, full-bleed canvas `3579 × 5031`).
- `poster-a3-rgb.png`: high-resolution RGB preview render.
- `poster-a3-cmyk.jpg`: CMYK export for print submission.
- `qr-github.png`: QR code pointing to the project GitHub repository.

Design notes:
- Uses the project's actual desktop web screenshot instead of a phone mockup.
- Keeps major text inside a safe zone larger than 5mm from trim edges.
- Adds a QR code so judges can open the repository quickly.
